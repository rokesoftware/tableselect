TableSelect

A jQuery plugin implementing table rows selection by mouse and keyboard comfortably.

Why?

Of course, there is a number of implementing the same kind of functionality as this one. But we needed something fairly light, mimicking the interface of Windows Explorer as much as possible with all the useful events fired... and writing it seemed easier (and more fun!) than bending anybody else's work.

Usage

It is a typical jQuery plugin built using the Widget Factoy, use it as such. On any selection change, select or unselect events are fired with some additional data that can be used to implement a variety of things. See the simple demo for the basic usage and the groups demo for something a bit more sophisticated.

Dependencies

It is a jQuery plugin, so you need jQuery and jQuery UI (widget at least).
